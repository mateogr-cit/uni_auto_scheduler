Backend Directory Structure (FastAPI)

backend/
├── main.py # app entry point, registers all routers
├── models.py # SQLAlchemy database models
├── database.py # DB connection and session config
└── routes/
├── auto_schedule.py # core scheduling algorithm
├── courses.py # course management
└── rooms.py # room management

Application package layout. The routes/ module groups all API endpoints by domain.
